# webapp-quickstart

Quickstart template for my webapp projects running with AngularJS 1.X (ES6), Webpack, Stylus, Jade

## Setup

```
git clone https://github.com/muratcorlu/webapp-quickstart.git myappname
cd myappname
rm -rf .git
APPNAME=myappname npm run setup
npm install
npm start
```
